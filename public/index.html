Timestamp Microservice

Enter a date in the input field below to get its Unix timestamp and UTC representation.

Supported formats:

  • Empty input (returns current time)
  • Unix timestamp (10 or 13 digit number)
  • ISO date string (e.g., "2023-01-01")
  • Any date format supported by new Date(date_string)

Try it out:

Example URLs:

  • /api/2015-12-25
  • /api/1451001600000
